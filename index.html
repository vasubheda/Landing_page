What?

This is a document classifier which classifies different type of documents which currently are Art and Science, Health, Politics and Government, Finance. It analyses the given input according to this categories and stacks them accordingly.

Why?

It is very difficult for a well built and dense organization to maintain and avoid information from mixing up, businesses are failing due to unstructured data and the most efficient way to prevent this from happening is to make use of AI tools and reduce time plus manual effort. This tool can be very helpful in sorting the information.

How



Web Scrapping


We use web scrapping (i.e. the process of using bots to extract the content and data from a website) to collect the data required to train the model and we specifically web scrapped the data from harward website.

Training


All the information we have collected from web scrapping is used to train the model to respond accordingly.

Deploy


It is uploaded online for the people to to use it as an open source api

Where?

The purpose of classification is to break down broad subjects into smaller, more manageable, more specific parts and this is where our document classification is applied. Businesses can use document classification apis to manage their day to day process .Our model can be used in organizational institutions to reduce man labour and unnecessary actions.
